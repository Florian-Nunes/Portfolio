Behind the scenes

I tried to create a timeline with alternating position for every even element.

As you can see, on wider screens you will get the elements positioned right and left next to the timeline. However I found other versions not perfect in terms of space management.

I didn't want to create a blank space on the other side of each entry. Using javascript you can create overlapping heights of the elements. In this example you see the following element overlapping the previous by 1/3 of it's height.
